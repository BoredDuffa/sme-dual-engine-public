# SME Dual Engine Public Dashboard

This repository hosts the public static Galway/Dublin SME Dual Engine dashboard.

The published files are generated from committed sample data only:

- `galway-dublin-sme-dashboard-sample.json`
- `galway-dublin-market-data-sample.csv`
- `galway-dublin-company-universe-sample.csv`

Do not commit private backend exports, private company lists, credentials, or
live client data to this public hosting repository.
